# E-Commerce Website - Proyek PKL

Website E-Commerce ini dikembangkan menggunakan framework Laravel sebagai bagian dari proyek Praktik Kerja Lapangan (PKL). Aplikasi ini bertujuan untuk memfasilitasi transaksi online dan manajemen produk secara digital.

## Teknologi yang Digunakan
* **Framework:** Laravel 11
* **Bahasa:** PHP & TypeScript (Inertia.js)
* **Database:** MySQL
* **Styling:** Tailwind CSS / Bootstrap

## Fitur Utama
* Manajemen Produk (CRUD)
* Keranjang Belanja (Shopping Cart)
* Sistem Autentikasi Pengguna
* Dashboard Admin

## Cara Menjalankan di Lokal
1. Clone repositori ini
2. Jalankan `composer install` dan `npm install`
3. Salin file `.env.example` menjadi `.env`
4. Jalankan `php artisan key:generate`
5. Jalankan `php artisan migrate`
6. Akses di browser melalui `php artisan serve`
